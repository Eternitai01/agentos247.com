## 🌐 Technology Stack

Built on **[OpenClaw](https://openclaw.ai)**—the open-source AI agent operating system trusted by enterprises worldwide.

### Core Infrastructure
- **OpenClaw Gateway**: Enterprise-grade agent orchestration and multi-channel routing
- **Hermes**: Advanced memory architecture with RAG and semantic search
- **GBrain**: Knowledge graph and long-term memory persistence
- **Multi-LLM Support**: Anthropic Claude, OpenAI GPT, Google Gemini, DeepSeek, and more
- **Containerized Deployment**: Docker-based isolation with automatic scaling
- **Extensible Skills**: Pre-built capabilities (calendar, email, web search, code execution) + custom agent skills
- **Persistent Memory**: Your agent remembers context, learns from interactions, and maintains continuity across conversations

### Agent Capabilities Powered By OpenClaw
- **Multi-channel messaging**: Telegram, WhatsApp, Signal, Slack, Discord
- **Voice & Audio**: TTS, voice commands, meeting transcription
- **Tool Integration**: Calendar (Google/Cal.com), email, file operations, web browsing
- **Code Execution**: Safe sandboxed shell access for automation
- **Cron Scheduling**: Time-based tasks, reminders, recurring workflows
- **Sub-agent Spawning**: Delegate complex work to specialized child agents
